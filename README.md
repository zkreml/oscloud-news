# Oscloud News

Statický blog / novinky pro projekt **Oscloud**.  
Oddělené od WordPressu, psané v Markdownu, generované do čistého HTML.

---

## Technologie

- **Hugo** (static site generator)
- **Markdown** (obsah)
- **Čisté HTML** (výstup)
- Hosting: **Surfer**
- Repo: **Gitea**

---

## Doména

Plánované umístění:

- `news.oscloud.cz`  
  nebo
- `blog.oscloud.cz`

Z hlavního webu `oscloud.cz` povede jen odkaz **Novinky**.

---

## Struktura projektu

```text
oscloud-news/
├─ content/
│  └─ posts/
│     └─ 2025-01-prvni-novinka.md
├─ layouts/
├─ static/
├─ config.toml
├─ public/
└─ README.md
```
